# IA Finance - Plataforma Financiera Inteligente

## 📋 Descripción
Plataforma de gestión financiera personal con 5 módulos innovadores basados en IA.

## 🚀 Módulos
- **DSE** - Decision Simulation Engine (Simulador financiero)
- **PFSE** - Prescriptive Financial Strategy (Estrategias con IA)
- **IOIE** - Income Opportunity Intelligence (Oportunidades de ingreso)
- **EIF** - Ecosystem Integration Fabric (Integración de servicios)
- **CFIL** - Conversational Financial Layer (Chat Roman)

## 🛠️ Tecnologías
- Frontend: Angular 17
- Backend: Node.js + Express
- Base de datos: MySQL
