# Despliegue Estático para Hostinger
Este directorio contiene una versión estática y autónoma de la página de login de IA Finance, lista para subir directamente a Hostinger.

## Instrucciones de uso
1. Sube TODO el contenido de la carpeta `public_html` a la raíz de tu hosting (carpeta `public_html`).
2. Asegúrate de que la imagen `logo_proyecto.png` esté en `assets/images/`.

## Nota
Esta versión es independiente del proyecto Angular en `/frontend`.
